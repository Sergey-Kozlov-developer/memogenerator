# App memogenerator

Учебный проект.

Приложение позволяет создавать мемы.
В нем реализованно:
- архитектура BLoC
- запросы к галерее телефона
- кнопка добавление мема
- создание шаблонов для мемов
- анимация при долгом нажатии на memogenerator
